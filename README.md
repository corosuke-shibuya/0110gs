①課題名
ブックマーク登録アプリ

②課題内容（どんな作品か）
	•	書籍/記事などのブックマークを「タイトル・URL・コメント」で登録できるアプリ
	•	登録したブックマークをDBから取得し、一覧ページで表形式で表示する

③アプリのデプロイURL（任意）
	•	https://corosuke-shibuya.sakura.ne.jp/a_php02/

④アプリのログイン用IDまたはPassword（ある場合）
	•	ID: なし
	•	PW: なし

⑤工夫した点・こだわった点
	•	DB保存時に日時（datetime）を自動で記録するようにし、フォーム入力なしで履歴が残るようにした
	•	一覧表示を <p> ではなく <table> で表形式に変更し、見やすく整理した
	•	URLをリンク化し、クリックで別タブ表示できるようにした（target=”_blank”）

⑥難しかった点・次回トライしたいこと（又は機能）
	•	さくらサーバーへのアップロードでフォルダが二重になったり、ローカルが誤操作で別のフォルダに移動したり、を繰り返して中々思ったように更新されず、心折れそうになりました。。。
	•	DBのテーブル定義とINSERT文のカラム名・必須項目が一致しておらず、エラー解消に苦戦した
	•	次回は「削除・更新機能」「入力チェック」「並び替え（新しい順）」も追加したい

⑦フリー項目（感想、シェアしたいこと等なんでも）
	•	phpでwebにアップするとコードにパスワードが書いててもわからないの不思議
